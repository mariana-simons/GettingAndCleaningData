Getting and Cleaning Data Specialization Course - Peer Assessments 

## Course Assignment Repo

This repo contains the following files:

 
* dataTidy.txt - a tidy data set which is an output of run_analysis.R
* run_analysis.R – R script which gets and cleans data from 

https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip 

* CodeBook.md - a code book that describes the variables and their values in the tidy data set, and the transformations the run_analysis.R script performs to clean up the data.
